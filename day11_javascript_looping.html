Looping

While

While loops are used to execute a block of code multiple times as long as a condiitonal is true.

Exemple:

        
            let i = 0;
            while (1 < 5) {
                console.log(1);
                i++;
            }

Do While Loop

Do while loops are used to execute a block of code at least once, and then reapeat the loop as long as a condition is true.

Exemple:

        
            let i = 0;
            do {
                console.log(i);
                i++

            }while (i < 5)

Output:

I call do while loops "Shoot first and ask questions later" loops , because they check the codition AFTER

You can simulate do while loop with regular while loop using break

For Loop

For loops are used to execute a block of code a number of times.

Exemple:

        
            for (let i = 0; i < 5; i++) {
                console.log(i);
            }

Switches

        
            let day = new Date().getDay();
            switch (day){
                case 0:
                console.log("Sunday");
                break;
                case 1:
                console.log("Monday");
                break;
                case 2:
                console.log("Tuesday");
                break;
                case 3:
                console.log("Wednesday");
                break;
                case 4:
                console.log("Thursday");
                break;
                case 5:
                console.log("Friday");
                break;
                case 6:
                console.log("Saturday");
