Styling html with CSS

There are 3 ways to add CSS to html:

  1. Inline - for costume adjustments (is more important than extrenal)
  2. Internal- tutorials or everything in one .html page (- if u have 1 page)
  3. External - for most pages (most used)

More on Internal and External CSS

Internal CSS is added to the head of the html document. It is used for smaller projects or tutorials

External CSS is added to a separate file. It is used for most projects.

Alternative to inline styles -use ID attribute

You can use id attributes to style a single element.

But you can use id only once per page. Well browser is not going to yell at you but if you duplicate id attribute, you are going to have a bad time
