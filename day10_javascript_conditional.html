Boolean values

Results of comparison operators are so called booelean values. They are named after George Boole

Boolean Truth table

A B A and B A or B not A
true true true true false
true false false true false
false true false true true
false false false false true

Assume we have 2 variables a and b they will hold truth values.

Let's say a = true and b = false

Then a AND b will be false

a OR b will be true

Not a will be false
